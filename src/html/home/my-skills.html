Мои навыки

  • HTML

    HTML (HyperText Markup Language) — это стандартизированный язык разметки, используемый для создания веб-страниц. Он определяет структуру и содержание веб-страницы, используя различные элементы (теги), которые указывают браузеру, как отображать контент. HTML позволяет добавлять текст, изображения, ссылки, таблицы, формы и другие элементы на веб-страницу.

  • CSS

    CSS (Cascading Style Sheets) — это язык стилей, используемый для описания внешнего вида и форматирования документа, написанного на языке разметки, таком как HTML. CSS позволяет отделить содержимое страницы (HTML) от её оформления, что упрощает разработку и поддержку веб-сайтов.

  • SASS

    Sass (Syntactically Awesome Style Sheets) — это препроцессор CSS, который расширяет возможности стандартного CSS, добавляя такие функции, как переменные, вложенные правила, миксины (mixins), импорт и наследование. Sass позволяет писать более структурированный и поддерживаемый код, упрощая работу с большими и сложными стилями.

  • JS

    JavaScript (JS) — это высокоуровневый, интерпретируемый язык программирования, который широко используется для создания интерактивных веб-страниц и веб-приложений. JavaScript работает на стороне клиента (в браузере) и может взаимодействовать с HTML и CSS для динамического изменения содержимого страницы, реагирования на действия пользователя и выполнения асинхронных запросов к серверу.

  • TS

    TypeScript (TS) — это строго типизированный язык программирования, который является надмножеством JavaScript. TypeScript добавляет статическую типизацию и другие возможности, такие как интерфейсы, перечисления и декораторы, к стандартному JavaScript.

  • BOOTSTRAP

    Bootstrap — это бесплатный и открытый фреймворк для разработки веб-сайтов и веб-приложений. Он включает в себя набор инструментов для создания адаптивных, мобильных веб-проектов. Bootstrap предоставляет готовые компоненты, такие как кнопки, формы, навигационные панели, модальные окна и многое другое, которые можно легко настроить и использовать в своих проектах.

  • REACT

    React — это JavaScript-библиотека для создания пользовательских интерфейсов, разработанная Facebook. React позволяет создавать сложные UI, разбивая их на маленькие изолированные части, называемые компонентами. Компоненты могут быть повторно использованы и объединены для создания более сложных интерфейсов. React использует виртуальный DOM для оптимизации обновлений интерфейса, что делает его очень эффективным.

  • GIT

    Git — это распределенная система контроля версий, разработанная для эффективного управления изменениями в исходном коде. Git позволяет разработчикам отслеживать изменения в файлах, координировать работу в команде, возвращаться к предыдущим версиям кода и многое другое.

  • WORDPRESS

    WordPress — это бесплатная и открытая система управления контентом (CMS), основанная на PHP и MySQL. Она широко используется для создания различных типов веб-сайтов, включая блоги, интернет-магазины, корпоративные сайты и многое другое. WordPress предоставляет гибкий и мощный инструментарий для создания и управления контентом, с возможностью расширения функционала через плагины и темы.
